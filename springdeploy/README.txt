Para ejecutar el proyecto empaquetado en Jar desde consola.
    -A través de Maven se realiza un package del proyecto. Esto creará un archivo jar en la carpeta target.
    -Ingresar a través de consola a la carpeta target hasta encontrar el archivo jar.
    -En la consola ejecutar el archivo Jar con: java -jar archivo.jar, esto ejecutará la aplicación de Spring y podrá recibir
    requests sin compilar todo nuevamente.
